Filters

Expressions can have filters, to alter the output quoting.

Functions can also have filters, eg for caching. (No function level filters are currently provided!). Its usually better to use a return expression.

Note that the * is picking up name as a parameter -- free variables (eg, not a param target, assigment, or loop iteration variable) will be added automatically. {builder::as_code('examples/filters.html')}
